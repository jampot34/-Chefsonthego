# Chefs on the Go

**Chefs on the Go** is a website for a chef recruitment and catering service.  
This is a front-end project built with HTML, CSS, and JavaScript.

## Features
- Responsive homepage
- Contact section
- Gallery of chefs or events
- Interactive elements via JavaScript

## Folder Structure
## How to View
1. Open `index.html` in your browser for local preview.  
2. Live site (once deployed with GitHub Pages): [https://jampot34.github.io/-Chefsonthego/](https://jampot34.github.io/-Chefsonthego/)

## How to Edit
- HTML → `index.html`  
- CSS → `styles/styles.css`  
- JavaScript → `scripts/script.js`  
- Images → `images/` folder
